# =============================================================================
# MAC-SPECIFIC SETUP NOTES
# =============================================================================
# 1. PYTHON VERSION: Use Python 3.10+ (preferably 3.11 or 3.12)
#    - Recommended: Use pyenv to manage Python versions
#    - For M-series Macs: Ensure Python is ARM-native, not x86 via Rosetta
#
# 2. HOMEBREW DEPENDENCIES:
#    brew install ffmpeg
#    brew install libsndfile
#    brew install libffi
#
# 3. XCODE COMMAND LINE TOOLS:
#    xcode-select --install
#    (Required for some binary package compilation)
#
# 4. INSTALLATION STEPS:
#    python -m venv venv
#    source venv/bin/activate
#    pip install --upgrade pip setuptools wheel
#    pip install -r requirements-mac.txt
#
# 5. VERIFY INSTALLATION:
#    python -c "import torch; print(f'PyTorch: {torch.__version__}')"
#    python -c "import cv2; print(f'OpenCV: {cv2.__version__}')"
#    python -c "import onnxruntime; print(f'ONNX Runtime: {onnxruntime.__version__}')"
#
# 6. PERFORMANCE NOTES:
#    - CPU processing will be slower than GPU (expect 3-5x slower for video analysis)
#    - Apple Silicon Macs (M1/M2/M3): PyTorch automatically uses Metal acceleration
#    - For better performance on Intel Macs: Consider using GPU via external NVIDIA GPU
#    - System will process videos sequentially on Mac (parallelization via Ray is still possible)
#
# 7. OLLAMA SETUP (for local LLM):
#    - Download from https://ollama.com
#    - Install and run: ollama serve
#    - Pull model: ollama pull llama2 (or llama3.2)
#    - Ensure Ollama is running before starting the backend
#
# 8. INDIAN LANGUAGE SUPPORT:
#    - Whisper handles 99 languages including all Indian languages
#    - Download the appropriate Whisper model: python -c "import whisper; whisper.load_model('base')"
#    - For better accuracy: Use 'small' or 'medium' models (larger, slower)
#
# 9. REMOVED DEPENDENCIES (GPU/NVIDIA specific):
#    - Removed: onnxruntime-gpu (using CPU version instead)
#    - Removed: pynvml (NVIDIA management library - not applicable to Mac)
#    - Removed: GPUtil (GPU utility - NVIDIA only)
#    - These were causing installation failures on Mac
#
# 10. TROUBLESHOOTING:
#    - If pip install fails for MediaPipe or OpenCV:
#      pip install --upgrade pip
#      pip install --no-cache-dir mediapipe opencv-python
#    - For M1/M2 Macs with binary issues:
#      ARCHFLAGS=-Wno-error=unused-command-line-argument-hard-error-in-future pip install <package>
